# anti-novel-domain
中文小说网站列表，用于做防火墙阻断用。

为防止内部网络中的用户浏览小说网站，收集了一些小说网站域名，用于做防火墙阻断。
## 用法
### OpenWrt + [Simple AdBlock](https://github.com/stangri/openwrt-simple-adblock)
OpenWrt 22.03
[Doc about Simple AdBlock](https://docs.openwrt.melmac.net/simple-adblock/)
